# Нежелательные расширения VS Code

Критерии отбора:  
- используют ИИ для помощи в написании/исправлении кода;  
- либо выполняют автоматический поиск ошибок, проблем в коде, стиле или доступности.

## Расширения с элементами ИИ

- `volartools.volar-ai@0.0.15` — AI-подсказки и помощь при разработке на Vue.

## Расширения для поиска ошибок / анализа кода

### PHP / backend

- `bmewburn.vscode-intelephense-client@1.16.1` — статический анализ PHP, подсветка ошибок и проблем.
- `devsense.intelli-php-vscode@0.12.17700` — расширенный PHP-анализ и подсказки.
- `devsense.phptools-vscode@1.62.18097` — комплексные инструменты PHP с диагностикой.
- `devsense.profiler-php-vscode@1.62.18097` — профилирование и поиск проблем производительности.
- `devsense.composer-php-vscode@1.62.18097` — анализ и контроль зависимостей Composer.
- `prisma.prisma@7.0.0` — проверки и диагностика схемы Prisma.

### JavaScript / HTML / CSS / доступность

- `dbaeumer.vscode-eslint@3.0.16` — линтер JS/TS, подсвечивает ошибки и нарушения стиля.
- `celianriboulet.webvalidator@1.3.1` — валидатор HTML/CSS/JS.
- `deque-systems.vscode-axe-linter@4.10.10` — анализ доступности (a11y), поиск проблем интерфейса.

### Стиль, орфография и форматирование

- `esbenp.prettier-vscode@11.0.0` — автоформатирование кода по заданным правилам.
- `streetsidesoftware.code-spell-checker@4.3.2` — поиск опечаток и ошибок в тексте.
